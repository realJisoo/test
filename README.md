# 심리테스트 사이트

모바일 우선으로 설계된 심리테스트 플랫폼입니다. 사용자 친화적인 UI/UX와 반응형 디자인을 제공합니다.

## 🌟 주요 기능

### 홈페이지 구성
- **상단바**: 사이트 아이콘, 햄버거 메뉴
- **메인 배너**: 드래그 가능한 이미지 슬라이더 (3개)
- **추천 콘텐츠**: 선별된 인기 테스트 (최대 3개)
- **모든 콘텐츠**: 등록된 전체 테스트 목록
- **푸터**: 사업자 정보, 연락처, 개인정보 처리방침

### 네비게이션
- **사이드메뉴**: 검색, 내정보 메뉴
- **검색 기능**: 팝업 검색창, 결과 페이지
- **페이지 이동**: 부드러운 전환 효과

### 테스트 시스템
- **MBTI T/F 테스트**: 12개 질문으로 구성된 심리테스트
- **진행률 표시**: 실시간 프로그레스 바
- **결과 공유**: 클립보드 복사, 네이티브 공유 API

## 📱 모바일 우선 디자인

- 최적화된 모바일 레이아웃 (최대 너비: 400px)
- 터치 친화적 인터페이스
- 반응형 그리드 시스템
- 스와이프 제스처 지원

## 🎨 디자인 특징

- **글래스모피즘**: 반투명 배경과 블러 효과
- **그라데이션**: 브랜드 컬러 기반 그라데이션
- **애니메이션**: 부드러운 호버 및 전환 효과
- **타이포그래피**: Noto Sans KR 폰트 사용

## 🚀 사용 방법

### 1. 기본 설정
1. 웹 서버에 모든 파일을 업로드
2. `images/` 폴더에 필요한 이미지 파일들 추가
3. `script.js`에서 콘텐츠 데이터 수정

### 2. 콘텐츠 관리
```javascript
// script.js의 contentList 배열에 새 테스트 추가
const contentList = [
    {
        id: 'new-test',                // 고유 ID
        title: '새로운 테스트',         // 제목
        description: '테스트 설명',     // 설명
        thumbnail: '🎯',               // 썸네일 이모지
        url: 'new-test.html',          // 링크 URL
        isRecommended: true            // 추천 콘텐츠 여부
    }
];
```

### 3. 푸터 정보 수정
사이트 관리자는 웹 페이지에서 직접 클릭하여 다음 정보를 편집할 수 있습니다:
- 사업자 정보
- 연락처 정보
- 개인정보 처리방침 링크

### 4. 배너 이미지 교체
`images/` 폴더의 배너 이미지들을 교체:
- `banner1.jpg` (권장 크기: 400×200px)
- `banner2.jpg`
- `banner3.jpg`

## 🔧 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 플렉스박스, 그리드, 애니메이션

- **Web APIs**: 클립보드 API, Share API
