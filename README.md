# 나는 대문자T? 대문자F? - MBTI 심리테스트

MBTI의 T/F를 재미 위주로 해석한 심리테스트 사이트입니다.

## 📋 프로젝트 개요

- **테스트명**: 나는 대문자T? 대문자F?
- **부제**: 나는 어떤 타입일까? 오열하는 MBTI F참가자? 너 T발 C야?
- **목적**: MBTI의 T/F를 재미 위주로 해석한 테스트 제공
- **타겟**: 10~20대 여성 (MBTI, 캐릭터테스트, 감성디자인, SNS공유 관심)

## 🎨 디자인 특징

- **스타일**: 픽셀아트 스타일 (부티나살롱, 뿌띠빠띠 스타일 참고)
- **해상도**: 모바일 기준 2340x1080px
- **참고 사이트**: 푸망, 방구석연구소
- **반응형**: 모바일 우선 디자인

## 📝 테스트 구성

### 질문 구성
- **총 질문 수**: 12개
- **질문 형태**: 이지선다형 (한 화면당 1문항)
- **점수 계산**: T 성향 +1점, F 성향 -1점

### 결과 유형
1. **매우 T형 (T)**: 점수 +5 ~ +12 - 로봇 그 자체
2. **비교적 T형 (t)**: 점수 0 ~ +4 - 감정이 있는 로봇
3. **비교적 F형 (f)**: 점수 -4 ~ -1 - 공감 중시
4. **매우 F형 (F)**: 점수 -12 ~ -5 - 감정 중요

## 🚀 사용법

1. `index.html` 파일을 웹 브라우저에서 열기
2. **이름을 입력**하고 "시작하기" 버튼 클릭 (이름은 선택사항)
3. 12개의 질문에 답하기
4. 결과 확인 및 공유

## 🎨 일러스트 삽입 방법

### 📸 이미지 사양

#### 메인 일러스트 (main-illustration.png)
- **파일명**: `main-illustration.png`
- **권장 사이즈**: 300px × 300px
- **권장 해상도**: 72dpi (웹용)
- **파일 형식**: PNG (투명 배경 지원)
- **파일 위치**: `images/` 폴더

#### 결과 일러스트 (result-illustration.png)
- **파일명**: `result-illustration.png`
- **권장 사이즈**: 400px × 400px
- **권장 해상도**: 72dpi (웹용)
- **파일 형식**: PNG (투명 배경 지원)
- **파일 위치**: `images/` 폴더

### 📋 삽입 단계

1. **이미지 준비**
   - 위 사양에 맞는 일러스트 제작
   - PNG 형식으로 저장 (투명 배경 권장)

2. **파일 배치**
   - `images/` 폴더에 이미지 파일 복사
   - 정확한 파일명으로 저장

3. **확인**
   - 브라우저에서 새로고침하여 이미지 표시 확인
   - 반응형 디자인에서 이미지 크기 조정 확인

### 🎯 디자인 팁

- **투명 배경**: PNG 형식으로 투명 배경을 사용하면 더 자연스러운 통합
- **픽셀아트 스타일**: 전체 디자인과 조화를 위해 픽셀아트 스타일 권장
- **색상 조화**: 배경 그라데이션과 조화되는 색상 사용
- **호버 효과**: 이미지에 마우스를 올리면 확대되는 효과 적용됨

## 📊 테스트 로직
